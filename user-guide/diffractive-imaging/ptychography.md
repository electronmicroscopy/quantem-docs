# Ptychography

Iterative ptychographic reconstruction, implemented in `quantem.diffractive_imaging.ptychography` with a simplified interface in `ptychography_lite`. Reconstructions are built from interchangeable model components (dataset, object, probe, detector, and origin models), with optional constraints and hyperparameter optimization.

Planned subsections, following the tutorial sequence (`ptycho_iter_01` to `ptycho_iter_08`):

1. Lite reconstructions: a minimal end-to-end reconstruction
2. Full reconstructions: the complete model interface
3. Constraints
4. Mixed-state and aberration-fitted probes
5. Multislice objects
6. Hyperparameter optimization
7. Workflow recommendations
8. Multi-GPU reconstructions
