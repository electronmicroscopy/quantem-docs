# Tomography

`quantem.tomography` implements tilt-series preprocessing and 3D reconstruction:

- [Reconstruction](./reconstruction.md): from a fast "lite" mode through full reconstructions, including preprocessing and alignment.

The pipeline follows the same lite/full pattern as ptychography, and HPC batch scripts are provided for large reconstructions. Runnable examples live in the [tomography tutorials](https://github.com/electronmicroscopy/quantem-tutorials/tree/main/tutorials/tomography).
