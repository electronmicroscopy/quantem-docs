# Citing quantEM

A quantEM methods paper is in preparation. Until it is published, please cite the software directly:

> quantEM Developers, *quantEM: quantitative electron microscopy analysis toolkit*, https://github.com/electronmicroscopy/quantem

```bibtex
@software{quantem,
  author = {{The quantEM Developers}},
  title  = {quantEM: quantitative electron microscopy analysis toolkit},
  url    = {https://github.com/electronmicroscopy/quantem},
  license = {MIT},
}
```

If your work relies on a specific algorithm implemented in quantEM (for example a particular ptychographic or tomographic reconstruction method), please also cite the original method paper. The relevant references are given in the docstrings and tutorial notebooks for each method.
